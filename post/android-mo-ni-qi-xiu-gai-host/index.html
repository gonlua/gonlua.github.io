Android模拟器修改host

2020-07-17

执行以下命令：

  • root获取root权限
    • adb root
  • 重新挂载
    • adb remount
  • 把hosts文件拉出来
    • adb pull /system/etc/hosts hosts
  • 修改hosts文件
  • 把hosts文件push回去
    • adb push hosts /system/etc/hosts

我到了这一步提示(Android版本是7.1.1)
adb: error: failed to copy 'hosts' to '/system/etc/hosts': couldn't create file: Read-only file system

解决方法(我的adb版本：1.0.36)：

  • adb push hosts /system/etc/hosts
  • adb root
  • adb shell
  • mount
  • exit
  • adb disable-verity
  • adb reboot
  • adb root
  • adb remount

现在就可以进行写操作了
