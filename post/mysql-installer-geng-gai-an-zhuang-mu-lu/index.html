MySQL installer更改安装目录

2020-10-23

用installer 1.4的时候，不能更改安装目录了，这里有个小技巧：
在默认安装的目录建立一个空文件夹，比如我安装的是MySQL Server 5.6 64bit，
就创建文件夹
C:\Program Files\MySQL\MySQL Server 5.6
installer检查到冲突，就可以修改安装路径了：
