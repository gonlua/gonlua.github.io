SpringBoot用Actuator的shutdown不能关闭进程的解决方法

2020-07-10

如果使用了scheduler（我用的是quartz），会阻止进程关闭。

解决方法是在Bean销毁前停掉scheduler:

@PreDestroy
public void preDestroy() {
    if (scheduler!=null) {
        try {
            scheduler.shutdown();
        } catch (SchedulerException e) {
            e.printStackTrace();
        }
    }
}
