create-react-app + Ant Design兼容IE11

2020-07-09 1 min read # antd # React
  1. 安装react-app-polyfill (我安装create-react-app后默认就装了)
  2. 在index.js最最上方添加
import "react-app-polyfill/ie11";
import "react-app-polyfill/stable";
  1. 在package.json中的browserslist中添加ie 11:
"browserslist": {
    "production": [
      ">0.2%",
      "not dead",
      "not op_mini all",
      "ie 11"
    ],
    "development": [
      "last 1 chrome version",
      "last 1 firefox version",
      "last 1 safari version",
      "ie 11"
    ]
  }
  1. 配置好后，Ant Design也支持了
